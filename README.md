# Education_Wage_Data_Modeling_Project
This repo contains the python code files and connected raw data files, along with a powerpoint deck to cover the key data insights. The CollegeScoreboard and Bureau of Labor Statistics are the sources of the data.

There are two code files. One contains the SQL analysis done using SQLite, and the other is the python portion of the analysis.

Also included are the raw data files pulled from the sources cited in the python code files.
